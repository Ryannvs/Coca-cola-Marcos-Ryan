O que é um repositório no Git e qual a sua importância?
É um repositório remoto. Ele é importante para poder acessar seus projetos de onde quer que esteja e também serve para trabalhos em equipe. Além disso, serve também para outras pessoas ou empresa acessar ou monitorar seus projetos, expandindo assim, a gama de novas oportunidades

Qual a função do Git Bash?
criar um repositório remoto

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
Git add adiciona um arquivo ou todos os arquivos no repositório remoto. Git arquivo adiciona um arquivo específico no repositório e git add . adiciona todos os arquivos no repositório

Por que utilizamos o ponto (.) no comando git add .?
para simbolizar que você quer adicionar todos os arquivos da pasta em que está

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
Git commit serve como uma linha do tempo, para fazer marcações importantes conforme você vai fazendo seu trabalho/projeto

Qual a função do comando git push?
enviar as atualizações para o repositório remoto e fazer commits

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
Fazer marcações importantes e também para o seu gestor/equipe saber onde você está ou quais foram as atualizações

Explique a função das tags HTML e como elas estruturam uma página.
Elas funcionam como uma estrutura para o site ou pág que está sendo criado. Ou seja, ela estrutura texto, títulos, imagens, subtítulos e etc.

O que são atributos em uma tag HTML e qual a sua função?
atributos são para mudar a tag, como por exemplo "color" que serve para mudar a cor

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
É um modo que ajuda a escrever os códigos com mais facilidade

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
define como a largura e altura de um elemento é feito. colocamos ele no inicio porque precisamos saber como os elementos serão ajustados na página

Por que é importante utilizar tags semânticas no HTML?
Sào tags que passam uma informação com um significado específico para o conteúdo

O que são media queries no CSS e qual a sua função?
É um módulo que definem expressões. Ele permite ajustar visualizações e uma variedade específica de dispositivos de saída sem mudar o conteúdo

Qual a função do GitHub Pages?
criar uma página para seu repositório

O que é o Flexbox no CSS e para que ele é utilizado?
Ele permite ajustar os elementos de forma fléxivel, ou seja, de todos os jeitos

Qual a diferença entre justify-content e align-items em um container Flexbox?
um ajusta no eixo principal e o outro no eixo transversal

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
define a capacidade de um item fléxivel crescer sem atrapalhar os outros itens dentro do container

Explique a diferença entre flex-direction: column; e flex-direction: row;.
Um ajusta em colunas e outro ajusta em linhas
